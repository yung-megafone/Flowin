Dynamic Modules

Best results use mods/manifest.json. Without a manifest, the launcher attempts a limited scan of known module paths where the browser allows local file checks.

Manifest Help

Create this file beside the launcher:

mods/manifest.json

Example:

{
  "modules": [
    {
      "id": "ibcl",
      "name": "IBCL",
      "version": "v2",
      "type": "Dashboard",
      "description": "Inbound carton loss and projection dashboard.",
      "path": "mods/IBCL.html",
      "icon": "fa-chart-line",
      "accent": "blue",
      "status": "Installed"
    },
    {
      "id": "shiftlog",
      "name": "ShiftLog",
      "version": "v1",
      "type": "Handoff",
      "description": "Flow Team shift handoff note generator.",
      "path": "mods/ShiftLog.html",
      "icon": "fa-clipboard-list",
      "accent": "green",
      "status": "Installed"
    }
  ]
}

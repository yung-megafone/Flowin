# ClipIt

ClipIt is a lightweight userscript designed to streamline ticket sharing by quickly copying a ticket title and URL in one action.

## Features
- Copies the ticket title and URL for easy sharing
- Provides a one-click button for fast copying
- Displays a toast notification to confirm the copy action

## Installation
1. Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/).
2. Clone this repository or download the ClipIt script.
3. Add the script to your userscript manager.
4. Open the supported ticketing interface and ClipIt will load automatically.

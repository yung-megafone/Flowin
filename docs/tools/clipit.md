# ClipIt

ClipIt is a lightweight userscript for copying ticket information in one action.

## Location

`tools/browser-userscripts/clipit/ClipIt.user.js`

## What it does

- Adds a quick-copy button to the supported ticketing page.
- Copies the ticket title and URL.
- Shows a toast/confirmation after copying.

## Install

1. Install Tampermonkey or another userscript manager.
2. Add `ClipIt.user.js` from this repo.
3. Open the supported ticketing interface.

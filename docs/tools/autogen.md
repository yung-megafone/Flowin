# AutoGen

AutoGen is a userscript for generating structured ticket title suggestions.

## Location

`tools/browser-userscripts/autogen/autogen.user.js`

## What it does

- Generates ticket title suggestions from entered issue/location details.
- Supports CA number/title formatting logic where applicable.
- Exists separately from ClipIt for now, but can later be merged if that becomes cleaner.

## What it doesn't

- Work.
